# WeRead Eye Care Extension

一个面向 `微信读书 Web` 的 Chrome 阅读提醒扩展。

它不使用摄像头，也不做眨眼检测。扩展只在你打开 `https://weread.qq.com/web/reader/*` 时工作，按“活跃阅读累计时间”提醒你休息。

## 功能

- 只在微信读书阅读页生效
- 最近 `3 分钟` 有滚动、点击、按键或滚轮操作时，才继续累计阅读时间
- 每累计 `20 分钟` 活跃阅读时间，弹出页内提醒
- 默认用 `TTS` 播报固定文案：`请休息一下，眨眼几次，再看远处十秒。`
- 提醒浮层会自动消失，不需要手动确认
- `popup` 只显示：
  - 今日阅读时长
  - 今日提醒次数
  - 阅读状态
  - 下次提醒
- `options` 只显示：
  - 今日阅读时长
  - 今日提醒次数
  - 阅读状态
  - 下次提醒
  - 导出 CSV
  - 清空本地统计
- 支持在 `popup` 里一键预览真实提醒浮层
- 统计只保存在 `chrome.storage.local`

## 当前没有

- 没有摄像头检测
- 没有眨眼率、恢复效果之类的视觉统计
- 没有提醒策略切换
- 没有自定义 TTS 文案
- 没有云同步

## 隐私

- 不请求摄像头权限
- 不上传阅读数据
- 不保存原始页面内容
- 不做云同步

## 快速开始

```bash
npm install
npm run build
```

然后在 Chrome 中：

1. 打开 `chrome://extensions`
2. 开启 `开发者模式`
3. 点击 `加载已解压的扩展程序`
4. 选择 `dist/`

## 使用方式

1. 打开微信读书阅读页
2. 正常阅读，扩展会自动累计活跃阅读时间
3. 到达 `20 分钟` 后，页面右上角会出现提醒浮层，并播报语音
4. 点击扩展图标，可以查看今日阅读、今日提醒、阅读状态和下次提醒
5. 如果你只想看提醒长什么样，直接点 `popup` 里的 `预览提醒`

## 当前边界

- 只支持微信读书阅读页
- 只做固定 `20 分钟` 活跃阅读提醒
- 默认只用 `TTS`，不再以普通提示音作为默认提醒
- 如果浏览器当前无法播报 `TTS`，仍会显示提醒浮层，但不会自动回退为提示音
- 当前 CSV 导出只包含日期、书名、阅读分钟数和提醒次数
