# Chrome Web Store Form Answers

## 权限与隐私表单

以下内容可直接填写到 Chrome Web Store 后台对应字段。

### 单一用途说明

```text
本扩展只有一个用途：在微信读书网页版中根据活跃阅读时间提醒用户休息，并显示本地阅读状态、下次提醒倒计时和提醒统计。
```

### 请求 storage 的理由

```text
storage 仅用于在当前浏览器本地保存提醒设置、阅读计时状态和本地统计数据，以便显示今日阅读、下次提醒倒计时，并支持导出 CSV。数据不会上传到远端服务器。
```

### 请求主机权限的理由

```text
主机权限仅用于在 https://weread.qq.com/web/reader/* 识别阅读页、读取当前书名与页面阅读活动，并在页面内显示提醒遮罩和播放本地提醒语音。扩展不会访问其他网站。
```

### 是否使用远程代码

选择：

```text
不，我并未使用远程代码
```

原因：

- 所有 JavaScript 都打包在扩展内
- 提醒语音是扩展内置音频文件
- 没有外部脚本、远程 Wasm、远程 eval

## 数据使用表单

按当前实现，建议这样勾选：

- 勾选 `用户活动`
- 勾选 `网站内容`
- 其他项都不勾选

### 为什么勾选用户活动

扩展会根据滚动、点击、按键、页面可见性和当前标签页状态判断是否处于活跃阅读。

### 为什么勾选网站内容

扩展会读取微信读书页面中的当前书名，用于本地统计和 CSV 导出。

### 为什么其他项不勾选

扩展不收集也不传输以下数据：

- 个人身份信息
- 健康信息
- 财务和付款信息
- 身份验证信息
- 个人通讯
- 位置
- 网络记录
- 音视频或图片内容

## 数据处理承诺

下列承诺项建议全部勾选：

- 我不会出于已获批准的用途之外的用途向第三方出售或传输用户数据
- 我不会为实现与我的产品的单一用途无关的目的而使用或转移用户数据
- 我不会为确定信用度或实现贷款而使用或转移用户数据
